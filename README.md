# 🪙 Gold Price Trend Analyzer (Tamil Nadu)

A modern, interactive web application to track **live gold prices**, analyze trends, compare metals, and make smarter investment decisions.

Built using **HTML, CSS, JavaScript**, and powered with **AI-based insights (Google Gemini API)**.

---

## 🚀 Features

### 📊 Live Gold Price Tracking
- Real-time **24K, 22K, 18K gold rates**
- Per gram & per pawn (Tamil Nadu standard)
- Auto-refresh with caching system

### 📈 Trend Analysis
- Interactive charts (7 days / 30 days / 90 days)
- Price movement insights
- Historical data visualization

### ⚖️ Metal Comparison
- Compare **Gold vs Silver vs Platinum**
- 30-day performance analysis

### 💰 Gold Rate Calculator
- Calculate gold value based on:
  - Karat (24K, 22K, 18K)
  - Units (Gram, Pawn, Tola, Kg, etc.)
- Optional:
  - GST (3%)
  - Making Charges (~12%)

### 💡 Investment Guidance
- Smart recommendations based on goals:
  - Long-term investment
  - Jewellery purchase
  - Trading
- Options include:
  - SGB (Sovereign Gold Bonds)
  - Gold ETF
  - Physical Gold

### 📉 SIP Planner
- Monthly investment calculator
- Shows:
  - Total investment
  - Expected returns
  - Profit

### 🤖 AI Insights (Gemini API)
- Live market analysis
- Ask questions like:
  - "Should I buy gold now?"
  - "Gold price prediction?"
- Real-time AI-powered responses

### 🌙 UI Features
- Dark / Light mode toggle
- Responsive design
- Clean premium UI

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Charts:** Chart.js
- **AI Integration:** Google Gemini API
- **Storage:** LocalStorage (for caching & settings)
